
# Unity 映像表現研究計画

## 概要

Unity を用いたリアルタイム映像表現の研究を行う。  
ゲーム開発における視覚表現をテーマとし、特に以下の観点から技術的実験を行う。

- シェーダー技術
- ポストプロセス
- プロシージャル生成
- ライティング
- GPUベースエフェクト

それぞれを小規模な技術デモとして実装し、表現手法と実装方法を比較する。

---

# 1. トゥーンレンダリング

## 目的

リアルタイムレンダリングにおけるアニメ調表現の実装手法を調査する。  
特に以下を検証する。

- 陰影の段階化
- 輪郭線表現
- スタイライズドライティング

---

## 方法

### 方法A：Ramp Texture によるトゥーンシェーディング

**概要**

ライトの内積を Ramp Texture に入力し、段階的な陰影を生成する。

**実装**

- N・L を計算
- Ramp Texture 参照
- Shader Graph または HLSL

**特徴**

- 実装が簡単
- アーティスト調整がしやすい

---

### 方法B：ステップ関数による陰影

**概要**

ライトの内積を step 関数で離散化する。

**実装**

```

float intensity = dot(normal, lightDir);
float shade = step(0.5, intensity);

```

**特徴**

- 数学的にシンプル
- GPU負荷が低い

---

### 方法C：カスタムライティングモデル

**概要**

Unity の標準ライティングを使わず、独自の BRDF を設計する。

**実装**

- URP/HDRP カスタムシェーダー
- N·L を非線形変換
- Rim light 追加

---

## 評価

以下の観点で比較する。

- 視覚品質
- 計算コスト
- 調整のしやすさ

---

# 2. ホログラム表現

## 目的

SF的なホログラム表現をリアルタイムシェーダーで実装する。

---

## 方法

### 方法A：Fresnel 発光

**概要**

視線方向と法線の角度に応じて発光を強くする。

**式**

```

fresnel = pow(1 - dot(viewDir, normal), power)

```

**効果**

- エッジ強調
- SF表現

---

### 方法B：スキャンライン

**概要**

UV座標に周期関数を適用し、走査線を作る。

**式**

```

scan = sin(uv.y * frequency + time)

```

---

### 方法C：ノイズによるグリッチ

**概要**

ノイズを使い頂点や色を揺らす。

**技術**

- Perlin Noise
- Simplex Noise

---

## 評価

- SF表現の説得力
- シェーダー負荷
- 組み合わせ可能性

# 3. VHS / CRT 表現

## 目的

アナログ映像風のポストプロセスを実装する。

---

## 方法

### 方法A：RGBチャンネル分離

**概要**

RGB の UV をずらす。

```

R = tex(uv + offset)
G = tex(uv)
B = tex(uv - offset)

```

---

### 方法B：ノイズ

画面に時間変化するノイズを加える。

---

### 方法C：画面歪み

UV を非線形変換する。

例：

```

uv = uv * (1 + distortion * r^2)

```

---

## 評価

- 映像のレトロ感
- GPU負荷
- エフェクト組み合わせ

# 4. プロシージャル地形

## 目的

ノイズ関数を用いた地形生成アルゴリズムを検証する。

---

## 方法

### 方法A：Perlin Noise

高さマップ生成

```

height = PerlinNoise(x, z)

```

---

### 方法B：Fractal Noise

複数スケールのノイズを合成

```

height = Σ noise(freq_i) * amp_i

```

---

### 方法C：侵食シミュレーション

地形生成後に erosion を適用する。

- Hydraulic erosion
- Thermal erosion

---

## 評価
